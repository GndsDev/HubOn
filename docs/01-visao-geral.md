# HubOn - Visão Geral

## Objetivo

Criar um sistema local para restaurantes, usando Angular, Spring Boot e PostgreSQL.

## Fluxo principal

Mesa → Comanda → Pedido → Itens do Pedido → Pagamento

## Módulos do MVP

- Login
- Usuários
- Mesas
- Categorias
- Produtos
- Comandas
- Pedidos
- Cozinha
- Pagamentos
- Relatórios básicos

## Fora do MVP

- Estoque
- Nota fiscal
- Delivery
- QR Code
- App mobile
- Multiempresa
