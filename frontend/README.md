# Frontend do HubOn

Aplicação Angular 21 do HubOn.

## Executar

```powershell
npm install
npm start
```

Acesse `http://localhost:4200`.

Para aceitar conexões da rede local:

```powershell
npm run start:network
```

## Testar e compilar

```powershell
npm test -- --watch=false
npm run build
```
